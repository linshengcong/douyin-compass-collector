# 抖音罗盘采集器使用说明

## 第一次使用

1. 解压收到的 ZIP，不要在压缩包内直接运行。
2. 确认电脑已安装 Google Chrome。
3. 双击打开 `抖音罗盘采集器`。
4. 首次启动会打开应用包内可见的 `采集器数据` 文件夹。
5. 将可信来源提供的 `.env` 文件重命名为 `配置.env`，放入 `采集器数据/配置.env`，然后重新启动应用。

程序会自动在 `采集器数据/runtime/` 保存 Chrome 登录态、采集日志、SQLite 数据和 CSV。`config/tasks.yaml` 也位于应用包内并随版本更新覆盖。

升级前请先复制备份 `.app/Contents/Resources/采集器数据`。Finder 替换整个 `.app` 时会同时替换包内数据；换成新版 `.app` 后，再把备份的 `采集器数据` 恢复到相同位置。

## 日常操作

1. 在桌面窗口中点击“开始采集”。
2. 首次使用时，在打开的 Chrome 中完成登录。
3. 采集完成后，在窗口中打开 CSV，或查看钉钉消息中的下载链接。

请勿将 `.env` 或 `配置.env` 发给不受信任的人，也不要将整个 `.app` 或 `采集器数据` 上传到 GitHub 或网盘公开链接。
