# 抖音罗盘采集器使用说明

## 第一次使用

1. 解压收到的 ZIP，不要在压缩包内直接运行。
2. 确认电脑已将 Google Chrome 正式版安装在系统标准位置；程序不内置 Chrome，不能用 Edge 或 Chromium 代替。
3. 双击打开 `抖音罗盘采集器`。
4. 首次启动会打开系统的应用数据文件夹。
5. 将可信来源提供的`配置.env`，放入该文件夹，然后重新启动应用。

程序会自动在应用数据目录的 `runtime/` 保存 Chrome 登录态、采集日志、SQLite 数据和 CSV。macOS 路径为 `~/Library/Application Support/抖音罗盘采集器/`；Windows 路径为 `%LOCALAPPDATA%/抖音罗盘采集器/`。`config/tasks.yaml` 位于应用包内并随版本更新覆盖。

升级前请先复制备份应用数据目录。替换 `.app` 不会影响该目录中的登录态和采集结果。

## 日常操作

1. 在桌面窗口中点击“开始采集”。
2. 首次使用时，在打开的 Chrome 中完成登录。
3. 采集完成后，在窗口中打开 CSV，或查看钉钉消息中的下载链接。

请勿将`配置.env` 发给不受信任的人。

桌面版只支持通过窗口操作；不要在命令行中运行 `login` 或 `run --no-gui`，它们需要终端输入。首次登录请在窗口中点击“开始采集”，在打开的 Chrome 中完成登录后，再次开始采集。
